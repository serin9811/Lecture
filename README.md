# Lecture

Lecture is a rest api written in Node.js

## How To Run

Download Node.js [here](https://nodejs.org/en/download/)

```
node app.js
```
## API List

# Lecture API

| Method | Path                                   | Description             |
| -------| -------------------------------------- | ----------------------  |
| GET    | /lecture/list?page=1&pageSize=10       | Read a list of lectures |
